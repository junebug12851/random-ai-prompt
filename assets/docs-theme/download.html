Download Random AI Prompt

Random AI Prompt ships as one codebase in two editions, and every one is available pre-built — nothing to compile unless you want to. Use it free in your browser, or download the full desktop app for your operating system. Neither is "the" version: the hosted site is just one deployment of the online edition, and you can self-host that same build or run the desktop app locally.

Use it online

Available now

The online edition runs entirely in your browser — bring your own API key (or just generate prompt text with no key). Nothing to install, nothing stored on a server. Want to run it yourself? Every release includes a static online.zip bundle you can host anywhere.

Desktop app

Available now

The full edition adds the image gallery, single-image view, an in-app content manager, local Stable Diffusion providers and an 18+ mode. It's self-contained — it bundles its own runtime, so there's nothing to install first. Pre-built for every platform:

  • Windows — installer (.msi / .exe) or a portable .zip
  • macOS — .dmg
  • Linux — .AppImage or .deb
