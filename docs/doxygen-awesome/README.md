# Vendored theme: doxygen-awesome-css

`doxygen-awesome.css` is vendored (copied in, not a submodule) so documentation
generation has **no network or submodule dependency** — it just works offline.

- Source: https://github.com/jothepro/doxygen-awesome-css
- License: MIT (see the SPDX header at the top of the `.css` file)
- Fetched from the `main` branch on 2026-06-06.

Referenced by the root `Doxyfile` via `HTML_EXTRA_STYLESHEET`. To update, replace
`doxygen-awesome.css` with a newer copy from the upstream repo. Do not hand-edit it.
