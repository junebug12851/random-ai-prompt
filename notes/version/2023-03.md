# Changelog — March 2023 (original build)

A short, practical month: **LoRA** support, a big batch of image-size **presets**, repo-hygiene fixes, and
a couple more dynamic prompts. 11 commits (2023-03-05 → 03-26).

> **Reconstructed retrospectively** from the git log, grouped **by theme** (predates the per-commit inline
> changelog rule — see [`../version.md`](../version.md)). Newest theme on top.

---

### 2023-03-26 — Cleanup, a futuristic prompt, and an upscale fix

Enhanced the prompt **cleanup** pass, added the **futuristic** dynamic prompt plus an old-fashioned
**extra-detail expansion**, and fixed a bug in the **upscale progress screen**.

### 2023-03-25 — Image-size presets + repo hygiene

Added **a large set of presets for different image sizes**, including wallpapers for common monitors —
regular, ultra-wide, and super-wide. On the housekeeping side: `update.bat` now ensures core files are
clean before `git pull`; **Windows shortcuts** and any file marked `-private`/`private-` are no longer
tracked; and a JSON issue in the presets was fixed.

### 2023-03-05 — LoRA support

Added support for **LoRA tags** in prompts, including inside **expansions** (even deeply nested, as usual),
so LoRA works seamlessly anywhere in a prompt.
